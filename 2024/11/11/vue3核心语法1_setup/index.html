vue3核心语法1_setup

十一月 11, 2024 vue框架 本文总阅读量次

vue框架学习笔记第二篇，笔记内容基于B站课程vue3+TypeScript。
本篇主要介绍vue3框架与vue2框架API的不同点与核心语法setup()。

OptionsAPI 与 CompositionAPI

  • vue2-OptionsAPI式（选项式）：数据、方法、计算属性等是分散在：data、methods、computed中的，若想修改一个需求，则需分别修改data、methods、computed，不便于维护和复用
  • vue3-CompositionAPI式（组合式）：通过 setup 函数，您可以自由地组织和复用逻辑，相同逻辑可以使用 composable 函数进行抽离。使用 ref 和 reactive 来创建响应式数据，能更好地控制数据传递和状态管理。

setup

setup概述

在 Vue3 中，setup() 函数是 Composition API 的核心部分，它用于声明组件的逻辑和响应式状态。在 setup() 中，所有的数据和方法都可以直接定义，而不需要再像 Vue2 中那样在不同选项（如 data、methods 等）中分散定义。

setup的返回值

setup返回值可以是一个函数，直接指定渲染的内容。

setup与OptionsAPI

二者可以共存，OptionsAPI可以通过this读取setup中的数据，setup()不可以读取data()、methods.

setup的语法糖

  2. 第二步：vite.config.ts
